# 🔥 FUBY Habit Tracker

A clean, offline-capable **Progressive Web App** for building daily habits and streaks.

---

## 🚀 Deploy to GitHub Pages

### Step 1 — Create a new GitHub repo
Go to [github.com/new](https://github.com/new), name it `fuby` (or anything you like).

### Step 2 — Upload these files
Make sure your repo contains:
```
fuby/
├── index.html
├── manifest.json
├── sw.js
├── icons/
│   ├── icon-192.png
│   ├── icon-512.png
│   └── screenshot.png
└── .github/
    └── workflows/
        └── deploy.yml
```

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select **GitHub Actions**
3. Push to `main` — the workflow deploys automatically

Your app will be live at:
```
https://YOUR_USERNAME.github.io/fuby/
```

---

## 📲 Installing on Android (Chrome)

1. Open the app in Chrome
2. Wait 5 seconds — an **Install App** sheet pops up automatically
3. Tap **Install Now** — done! Opens like a real app, no browser bar

## 📲 Installing on iPhone (Safari)

1. Open the app in **Safari** (not Chrome)
2. Tap the **Share** button (box with arrow at bottom)
3. Tap **"Add to Home Screen"** → **Add**

## ✨ Features

- Today view with progress bar
- Habits manager with 30-day progress
- Weekly calendar grid
- Stats — streaks, completions, consistency
- Dark & Light mode
- Works 100% offline
- All data stored locally on your device
